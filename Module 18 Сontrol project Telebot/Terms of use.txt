@Currencies_Spring_Bot

Это телеграм-бот для конвертации валют.

Чтобы его запустить, нужно скачать проект, перед этим установить пакеты requests и pytelegrambotapi, 
и запустить модуль Currencies_telebot_app2.
Затем открыть в телеграме бот по имени @Currencies_Spring_Bot и следовать инструкциям, начиная с команды /start.

Выполнено в качестве итогового задания по разделу Python в рамках курса "Тестировщик-автоматизатор на Python (QAP)".
